## 金融术语表
### 订单簿相关
| 英文术语 | 中文释义 | 说明 |
|------------|------------|------|
| Order Book | 订单簿 | 展示所有未成交的买单（Bid）和卖单（Ask）的列表 |
| Bid | 买单/买价 | 买方愿意出的最高价格 |
| Ask / Offer | 卖单/卖价 | 卖方愿意接受的最低价格 |
| Spread | 价差 | 最优卖价（Ask）与最优买价（Bid）之间的差额 |
| Depth | 深度 | 订单簿上在不同价格档位的挂单数量，反映市场流动性 |
### 订单类型
| 英文术语 | 中文释义 | 说明 |
|------------|------------|------|
| Limit Order | 限价单 | 以指定价格买入或卖出，只有达到该价格才会成交 |
| Market Order | 市价单 | 以当前市场最优价格立即买入或卖出 |
| Stop Order | 止损单 | 达到某个触发价后，自动转为市价单或限价单 |
| Fill or Kill (FOK) | 全部成交或全部取消 | 要么全部成交，要么全部取消 |
| Immediate or Cancel (IOC) | 立即成交或取消 | 能成交的部分立即成交，剩余部分自动取消 |
| Good Till Cancelled (GTC) | 撤销前有效单 | 挂单一直有效，直到被成交或主动撤销 |
| Post Only | 只挂单 | 只作为挂单进入订单簿，不会立即与已有订单成交 |
| Iceberg Order | 冰山单 | 只显示部分挂单数量，隐藏大额订单真实规模 |
### 撮合与成交
| 英文术语 | 中文释义 | 说明 |
|------------|------------|------|
| Match | 撮合 | 买单和卖单在价格上达成一致并成交的过程 |
| Taker | 吃单者 | 以市场已有挂单成交的人（主动成交，通常是市价单） |
| Maker | 挂单者 | 在订单簿上挂单等待成交的人（被动成交，通常是限价单） |
| Partial Fill | 部分成交 | 一个订单只成交了部分数量，剩余部分继续挂单 |
| Full Fill | 完全成交 | 一个订单全部成交 |
| Order ID | 订单ID | 每个订单的唯一标识 |
| Self-Trade | 自成交 | 同一账户的买单和卖单互相成交 |
### 资产与结算
| 英文术语 | 中文释义 | 说明 |
|------------|------------|------|
| Base Currency | 主币 | 交易对左边的币种（如 BTC/USDT 中的 BTC） |
| Quote Currency | 报价币 | 交易对右边的币种（如 BTC/USDT 中的 USDT） |
| Balance | 余额 | 用户账户中可用的资产数量 |
| Frozen/Locked Balance | 冻结余额 | 已经挂单但尚未成交的部分，被锁定不能用 |
| Settlement | 结算 | 成交后资产的实际转移过程 |
| Custody | 托管 | 资产由第三方（如交易所）保管 |
| Withdrawal | 提币 | 将资产从交易所提到链上钱包 |
| Deposit | 充值 | 将资产从链上钱包转入交易所 |
### 价格与行情
| 英文术语 | 中文释义 | 说明 |
|------------|------------|------|
| Last Price | 最新成交价 | 最近一笔成交的价格 |
| Mark Price | 标记价格 | 用于合约等衍生品的参考价格，防止恶意操纵 |
| Index Price | 指数价格 | 由多个市场加权计算的价格 |
| High/Low | 最高/最低价 | 一定时间内的最高价和最低价 |
| Volume | 成交量 | 一定时间内成交的总数量 |
| Open/Close | 开盘/收盘价 | 某一时间段的第一个/最后一个成交价 |
| VWAP | 成交量加权平均价 | 以成交量加权的平均价格 |
| TWAP | 时间加权平均价 | 以时间加权的平均价格 |
### 其他常用术语
| 英文术语 | 中文释义 | 说明 |
|------------|------------|------|
| Slippage | 滑点 | 实际成交价与下单时预期价格的差异 |
| Liquidity | 流动性 | 市场买卖的活跃程度，流动性高表示容易成交且价差小 |
| Fee | 手续费 | 交易所收取的服务费用，通常分为 Maker Fee 和 Taker Fee |
| Tick Size | 最小价格变动单位 | 价格可以变动的最小单位 |
| Lot Size | 最小下单单位 | 可以下单的最小数量 |
| Minimum Order Size | 最小订单量 | 允许下单的最小数量 |
| Order Expiry | 订单过期 | 订单在指定时间后自动失效 |
区块链/DEX 特有
| 英文术语 | 中文释义 | 说明 |
|------------|------------|------|
| On-chain | 链上 | 所有操作和数据都在区块链上完成 |
| Off-chain | 链下 | 某些操作在链下完成，最终结果再上链 |
| AMM (Automated Market Maker) | 自动做市商 | 如 Uniswap，使用算法自动提供流动性和定价 |
| CEX (Centralized Exchange) | 中心化交易所 | 如 Binance、Coinbase |
| DEX (Decentralized Exchange) | 去中心化交易所 | 如 Uniswap、Serum |
| Gas Fee | 矿工费/手续费 | 区块链上每笔交易需要支付的费用 |
| Nonce | 随机数/计数器 | 防止交易重放攻击的计数器 |
| Smart Contract | 智能合约 | 部署在区块链上的自动执行代码 |
| Liquidity Pool | 流动性池 | AMM中用户存入资产以提供流动性的资金池 |
| Impermanent Loss | 无常损失 | AMM中因价格波动导致的流动性提供者损失 |
| Oracle | 预言机 | 为区块链提供链外数据（如价格）的服务 |
### 衍生品与杠杆
| 英文术语 | 中文释义 | 说明 |
|------------|------------|------|
| Derivative | 衍生品 | 基于现货资产价格的金融合约，如期货、期权 |
| Futures | 期货 | 约定未来以特定价格买卖资产的合约 |
| Perpetual Swap | 永续合约 | 没有到期日的期货合约 |
| Option | 期权 | 在未来某一时间以特定价格买入或卖出资产的权利 |
| Leverage | 杠杆 | 用较小资金撬动更大交易额的机制 |
| Margin | 保证金 | 用于杠杆交易的抵押资产 |
| Liquidation | 强制平仓 | 保证金不足时，系统自动平掉仓位 |
| Funding Rate | 资金费率 | 永续合约多空双方定期支付的费用 |
| Mark-to-Market | 按市价计价 | 每日根据市场价格调整合约价值 |
### 风险管理与风控
| 英文术语 | 中文释义 | 说明 |
|------------|------------|------|
| Risk Limit | 风险限额 | 限制单个账户或合约的最大风险敞口 |
| Circuit Breaker | 熔断机制 | 市场剧烈波动时自动暂停交易 |
| Stop Loss | 止损 | 价格达到某一水平时自动平仓以限制损失 |
| Take Profit | 止盈 | 价格达到某一水平时自动平仓以锁定利润 |
| Auto-Deleveraging | 自动减仓 | 极端情况下，系统自动减小高杠杆用户仓位 |
### 其它
| 英文术语 | 中文释义 | 说明 |
|------------|------------|------|
| KYC (Know Your Customer) | 实名认证 | 金融机构对客户身份的验证流程 |
| AML (Anti-Money Laundering) | 反洗钱 | 防止非法资金流入金融系统 |
| Whale | 大户 | 持有大量资产、能影响市场的大型投资者 |
| Arbitrage | 套利 | 利用不同市场或产品间的价格差获利 |
| Flash Loan | 闪电贷 | 无需抵押、在一个区块内完成的超大额借贷 |
| Rug Pull | 地毯式跑路 | DeFi项目方卷款跑路的骗局 |
| Airdrop | 空投 | 项目方免费发放代币给用户 |
| Staking | 质押 | 锁定资产以获得奖励或参与网络共识 |
| Fork | 分叉 | 区块链协议升级或分裂为两个链 |
